# 03. Rules, Eligibility, Submission

> 기준: https://chain.link/hackathon/faq + prizes 페이지 공통 요구사항

## 1) 참가 자격

- 전 세계 참가 가능
- 단, U.S. 제재국 기반 참가자는 제외
- 팀 구성: **1~5명**
- 참가비: 무료

## 2) 기술/경험 기대치

- 스마트컨트랙트 및 블록체인 경험 권장
- CRE는 프로덕션 지향 환경으로, 초급자 친화형 이벤트는 아님
- TypeScript 경험이 사실상 중요

## 3) 프로젝트 적격 요건

1. **CRE를 의미 있게 사용해야 함**
2. 해커톤 기간(2/6 이후) 작업분이 핵심
3. 기존 프로젝트 확장은 가능하나 신규 작업분 증빙 필요
4. 과거 동일 제출물 재출품은 불가(유의미한 신규 구성 필요)

## 4) 제출 필수물

- 공개 소스코드(예: public GitHub)
- 3~5분 공개 데모 영상
- README에 Chainlink 사용 파일/구조 명시
- CRE workflow의 실행 또는 시뮬레이션 증빙

## 5) 프론트엔드/호스팅 요건

- 프론트엔드 **필수 아님**
- 외부 배포 **필수 아님** (로컬 데모 가능)

## 6) 심사 기준 (FAQ 요약)

- Technical execution
- Blockchain technology application
- Effective use of CRE
- Originality (wow factor)

## 7) 수상 후

- 상금 수령 시 KYC 등 개인정보 제출 필요
- IP는 팀이 보유(FAQ 기준)
