# 00. Overview

## 1) 이벤트 한 줄 요약

**Convergence**는 Chainlink의 CRE(Chainlink Runtime Environment) 기반 글로벌 온라인 해커톤입니다.

- 공식 페이지: https://chain.link/hackathon
- 기간: Feb 6 – Mar 1, 2026 (ET)
- 제출 마감: Mar 1, 11:59 PM ET

## 2) 지금 당장 중요한 5가지

1. **CRE를 의미 있게 사용**해야 상금 심사 대상입니다.
2. 프로젝트는 원칙적으로 **해커톤 시작 후 작업분**이어야 합니다.
3. **데모 영상(3~5분)은 필수**입니다.
4. **공개 코드 리포 + README**가 필수입니다.
5. 팀원은 각자 별도 등록해야 합니다.

## 3) 참가 형태

- 완전 온라인
- 커뮤니케이션: Discord
- 워크숍: YouTube
- 제출: Airtable 제출 폼

## 4) 빠른 액션 플랜

- D-6 ~ D-5: 트랙 선정 + CRE 최소 동작
- D-4 ~ D-3: 기능 1개 확실히 완성
- D-2: README + 구조 정리
- D-1: 데모 촬영/편집
- D-day: 제출 + 링크 최종 검증

## 5) 리스크 포인트

- “CRE 사용은 했지만 프로젝트 핵심이 아닌 경우” 감점/탈락 위험
- 5분 초과 데모 영상
- README에서 Chainlink 관련 파일/아키텍처가 불명확한 경우
- 기존 프로젝트 재활용 시 해커톤 기간 작업분 증빙 부족
